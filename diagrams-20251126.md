# ResourceDescriptor Building Flow Diagrams

I'll create comprehensive diagrams to illustrate how ResourceDescriptors are built for different plugins in the Adapt system. These diagrams show the discovery process, plugin-specific loading mechanisms, and the resulting data structures.

## Sequence Diagram: Resource Discovery and Descriptor Building

```mermaid
sequenceDiagram
    participant Discovery as discover_resources()
    participant Config as AdaptConfig
    participant PluginFactory as Plugin Factory
    participant Plugin as Plugin Instance
    participant Descriptor as ResourceDescriptor
    participant Companion as Companion Files

    Discovery->>Config: get_plugin_factory(ext)
    Config-->>Discovery: plugin_cls
    Discovery->>PluginFactory: plugin_cls()
    PluginFactory-->>Discovery: plugin instance

    Discovery->>Plugin: load(path)
    alt Single Descriptor (HTML, Markdown, Python)
        Plugin-->>Discovery: ResourceDescriptor
        Discovery->>Discovery: descriptors = [loaded]
    else Multiple Descriptors (Excel sheets)
        Plugin-->>Discovery: Sequence[ResourceDescriptor]
        Discovery->>Discovery: descriptors = loaded
    end

    loop For each descriptor
        Discovery->>Discovery: Calculate paths
        Discovery->>Descriptor: Set schema_path, ui_path
        Discovery->>Plugin: generate_companion_files(descriptor)
        Plugin->>Companion: Create schema.json, index.html
        Discovery->>Discovery: Create DatasetResource
    end
```

## Detailed Sequence: DatasetPlugin ResourceDescriptor Building

```mermaid
sequenceDiagram
    participant DatasetPlugin
    participant ResourceDescriptor
    participant Metadata

    DatasetPlugin->>DatasetPlugin: _get_header_and_sample(path)
    DatasetPlugin-->>DatasetPlugin: header, sample
    
    DatasetPlugin->>ResourceDescriptor: ResourceDescriptor(path, resource_type)
    ResourceDescriptor-->>DatasetPlugin: descriptor
    
    DatasetPlugin->>Metadata: descriptor.metadata["header"] = header
    DatasetPlugin->>Metadata: descriptor.metadata["sample_row"] = sample
    DatasetPlugin->>Metadata: descriptor.metadata["primary_key"] = "_row_id"
    
    DatasetPlugin-->>DatasetPlugin: descriptor
```

## Detailed Sequence: ExcelPlugin Multi-Sheet ResourceDescriptor Building

```mermaid
sequenceDiagram
    participant ExcelPlugin
    participant Workbook
    participant Sheet
    participant ResourceDescriptor
    participant Descriptors

    ExcelPlugin->>Workbook: load_workbook(path)
    Workbook-->>ExcelPlugin: workbook
    
    ExcelPlugin->>Workbook: workbook.sheetnames
    Workbook-->>ExcelPlugin: sheet_names
    
    loop For each sheet_name
        ExcelPlugin->>Sheet: workbook[sheet_name]
        Sheet-->>ExcelPlugin: sheet
        
        ExcelPlugin->>Sheet: iter_rows header & sample
        Sheet-->>ExcelPlugin: header_row, sample_row
        
        ExcelPlugin->>ExcelPlugin: _ensure_header(header_row)
        ExcelPlugin-->>ExcelPlugin: normalized_header
        
        ExcelPlugin->>ResourceDescriptor: ResourceDescriptor(path, "excel")
        ResourceDescriptor-->>ExcelPlugin: descriptor
        
        ExcelPlugin->>ResourceDescriptor: metadata["header"] = normalized_header
        ExcelPlugin->>ResourceDescriptor: metadata["sample_row"] = normalized_sample
        ExcelPlugin->>ResourceDescriptor: metadata["primary_key"] = "_row_id"
        ExcelPlugin->>ResourceDescriptor: metadata["sub_namespace"] = sheet_name
        
        ExcelPlugin->>Descriptors: descriptors.append(descriptor)
    end
    
    ExcelPlugin->>Workbook: close()
    ExcelPlugin-->>ExcelPlugin: descriptors
```

## Metadata Structure by Plugin Type

| Plugin Type | Metadata Keys | Description |
|-------------|---------------|-------------|
| **CSV** | `header`, `sample_row`, `primary_key` | Column headers, sample data row, primary key field |
| **Excel** | `header`, `sample_row`, `primary_key`, `sub_namespace` | Same as CSV plus sheet name for multi-sheet files |
| **HTML** | None | No additional metadata needed |
| **Markdown** | None | No additional metadata needed |
| **Python** | None | No additional metadata needed |
